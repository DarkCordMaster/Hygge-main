# Hegge - 크라우드 펀딩 플랫폼

개발 기간  
2024.12 ~ 2025.03

프로젝트 형태  
팀 프로젝트 (4인)  
개인 기여도: 40% 이상

---

## 🛠 기술 스택

- Frontend: React, JavaScript
- Backend: Spring Boot, Java
- Database: Oracle
- Infra / Tool: AWS, Git, Figma

---

## 💡 주요 기능

- 펀딩 프로젝트 등록 및 관리
- 결제 시스템 연동
- 실시간 펀딩 현황 대시보드
- 회원 관리 시스템

---

## 📌 담당 업무

- 프로젝트 아키텍처 설계 및 DB 모델링
- 마이페이지 / 인포메이션 화면 개발
- 로그인, 아이디·비밀번호 찾기 기능 구현
- 1:1 문의 게시판 기능 구현
- 커뮤니티 기능 개발
- 회원가입 및 사용자 유형별 화면 구성

---

## 🎯 성과

- 기획부터 개발까지 전체 프로젝트 경험
- 결제 시스템 연동을 통한 트랜잭션 처리 및 예외 대응 경험
- 협업을 통한 일정 조율 및 역할 분담 경험
- UI/UX 개발 참여로 사용자 중심 화면 설계 경험
- MyBatis 기반 서비스 계층 설계 경험

---

## 📚 배운 점

- ERD 설계를 통한 데이터 정규화 및 관계 설정 이해
- 팀 프로젝트에서의 협업 및 커뮤니케이션 중요성 인식
- 서비스 계층 분리를 통한 유지보수성과 확장성 고려
